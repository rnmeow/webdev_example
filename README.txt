webdev_example
==============

This is a repository for simply demo Web frontend / backend interaction.

* Frontend

Standalone HTML file, including JavaScript with Fetch API codes.

* Backend

Python Flask app, install dependencies:

	pip install -r requirements.txt

then run development server by executing:

	python main.py

In my environment, the server appears at <http://localhost:5000>.
