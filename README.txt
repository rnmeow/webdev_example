webdev_example
==============

This is a repository for simply demo Web frontend / backend interaction.

* Frontend

Standalone HTML file, including JavaScript with Fetch API codes.

* Backend

Written in Go, using `net/http` in standard library.

Install the Go compiler (<https://go.dev/dl/>) if you didn't.
Then, start the server by executing:

	go run .

The server should appear at <http://localhost:8080>.
